A Common Lisp dialect
that fits in your head.

ren is five reader-macros, thirty lines of code, and one rule: the subject is implicit. 

;; methods read like prose — `i` is self, `$x` is its slot
(defmethod add ((i num) v)
  (incf $n)
  (let ((d (- v $mu)))
    (incf $mu (/ d $n))
    (incf $m2 (* d (- v $mu)))))

;; chain through structures, look up config, bind `it`
(when (< (? data cols n) !budget)
  (aif (next-row pool)
       (label it)))

Why ren?

Tiny

The whole language is one file: a reader-macro definer plus four reader macros. Drop it in, (load "lib"), and you're done.

Anaphoric

In a method, i is always self. In an aif, it is always the test. No more boilerplate let-bindings.

Composable

$, !, and ? read as Lisp forms. They setf, they incf, they nest — they're invisible until you look.

Try it

Two files. No build system, no package manager, no asdf:load-system. If you have CLISP or SBCL, you have ren.

$ git clone https://github.com/timm/lisp
$ cd lisp
$ sbcl --script fri2.lisp --stats
$ clisp fri2.lisp --tree

The five macros

!key

Config lookup.

Reads from *the*, an alist of (name value flag doc) tuples. The flag and doc serve double duty as the command-line spec.

!budget   ;; ≡   (second (assoc 'budget *the*))

$field

Self slot access.

Inside a method whose first parameter is named i, $field expands to (slot-value i 'field). Works with setf, incf, and friends.

(defmethod mid ((i num)) $mu)

(defmethod add ((i sym) v &optional (w 1))
  (unless (eql v '?)
    (incf $n w)))

(? x a b c)

Nested slot access.

Chains slot-value calls. setf-able. Replaces the noisy (slot-value (slot-value ...)) tower.

(? data cols y)
;; ≡
(slot-value (slot-value data 'cols) 'y)

(aif test then else)

Anaphoric if.

Binds the test result to it in both branches. Borrowed from Paul Graham's On Lisp.

(aif (find flag opts :key #'third :test #'equalp)
     (setf (second it) (thing arg)))

(defread c (s) body)

Reader-macro definer.

Installs a one-character reader macro. The other four macros are all built with this. Six lines of code, and the language extends itself.

(defread $(s)
  `(slot-value i ',(read s t nil t)))

(defread !(s)
  `(second (assoc ',(read s t nil t) *the*)))

A complete example

Here is a streaming numeric summary — mean and standard deviation in constant memory, using Welford's method — written in plain Common Lisp on the left, and in ren on the right.

Plain Common Lisp

(defmethod add ((self num) v)
  (incf (slot-value self 'n))
  (let ((d (- v (slot-value self 'mu))))
    (incf (slot-value self 'mu)
          (/ d (slot-value self 'n)))
    (incf (slot-value self 'm2)
          (* d (- v (slot-value self 'mu))))))

ren

(defmethod add ((i num) v)
  (incf $n)
  (let ((d (- v $mu)))
    (incf $mu (/ d $n))
    (incf $m2 (* d (- v $mu)))))

Same code, half the noise, no new concepts. ren doesn't hide Lisp — it gets out of the way.
