Install

Two files. No build system, no package manager. Any CL with defstruct and defmethod — SBCL, CLISP, ECL, CCL — will run it.

Get the code

$ git clone https://github.com/timm/lisp
$ cd lisp

Run the example program

fri2.lisp is a complete active-learning system in ~250 lines, written in plus. It loads plus.lisp and runs from any CL.

$ sbcl --script fri2.lisp --stats
$ clisp fri2.lisp --tree
$ ./fri2.lisp --active            ; via shebang

Use plus in your own file

(load "plus")

(plus stat "Online mean."
    ((n 0) (mu 0))
  (add (v)
    (incf $n)
    (incf $mu (/ (- v $mu) $n))))

(let ((s (make-stat)))
  (dolist (v '(1 2 3 4 5)) (add s v))
  (format t "mean=~a~%" (? s mu)))
; mean=3
