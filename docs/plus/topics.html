Manual

Eleven forms. Each gets a section.

$field

Self slot access.

Inside a method whose first parameter is named i — plus always names it that — $x expands to (slot-value i 'x). Read, setf, incf, push all work.

Unexpanded
$mu
(incf $n)
(setf $has nil)
Expanded
(slot-value i 'mu)
(incf (slot-value i 'n))
(setf (slot-value i 'has) nil)

(? x a b c)

Nested slot access.

Chained slot-value. setf-able. Replaces the noisy (slot-value (slot-value ...)) tower.

Unexpanded
(? data cols y)

(setf (? tree left) new-left)
Expanded
(slot-value
  (slot-value data 'cols) 'y)

(setf (slot-value tree 'left)
      new-left)

(! f a b)

Funcall shortcut.

Equivalent to (funcall f a b). Three chars shorter, reads cleaner when functions ride in arguments.

Unexpanded
(! cmp a b)
(mapcar (f+ (! key _)) lst)
Expanded
(funcall cmp a b)
(mapcar (lambda (_)
          (funcall key _))
        lst)

f+ / ff+ / fff+

Lambda shortcuts.

One-, two-, three-arg lambdas with implicit parameters _, __, ___. Kills the (lambda (x) ...) overhead in higher-order calls.

Unexpanded
(mapcar (f+ (* _ _))
        '(1 2 3))

(mapcar (ff+ (+ _ __))
        '(1 2) '(10 20))

(sort lst #'<
  :key (f+ (? _ at)))
Expanded
(mapcar (lambda (_) (* _ _))
        '(1 2 3))

(mapcar (lambda (_ __)
          (+ _ __))
        '(1 2) '(10 20))

(sort lst #'<
  :key (lambda (_)
         (slot-value _ 'at)))

(if+ test then else)

Anaphoric if.

Binds the test result to it in both branches. From Paul Graham's On Lisp.

Unexpanded
(if+ (gethash key cache)
     it
     (setf (gethash key cache)
           (compute key)))
Expanded
(let ((it (gethash key cache)))
  (if it
    it
    (setf (gethash key cache)
          (compute key))))

(let+ ...)

Sequential bindings: let, dbind, mvb, labels.

One construct, four forms. Entry shape picks: plain (var val) is let; flat-symbol list ((a b c) form) is multiple-value-bind; nested-pattern list is destructuring-bind; 3+ elements (name args body...) is labels.

Unexpanded
(let+ ((x       5)
       ((a b c) (values 1 2 3))
       (sq      (n) (* n n))
       (((l r) p) '((1 2) 3)))
  (sq (+ x a)))
Expanded
(let ((x 5))
  (multiple-value-bind (a b c)
      (values 1 2 3)
    (labels ((sq (n) (* n n)))
      (destructuring-bind
          ((l r) p) '((1 2) 3)
        (sq (+ x a))))))

Replaces nested let* / multiple-value-bind / destructuring-bind / labels stairs with one flat block. let* becomes redundant.

(for+ expr for x in lst if test ...)

List comprehension.

Reads like Python. Walks for and if clauses, collecting expr per surviving iteration. Nil results are skipped — a guard expression doubles as filter.

Unexpanded
(for+ (* x x)
  for x in '(1 2 3 4 5)
  if (oddp x))
Expanded
(loop for x in '(1 2 3 4 5)
  append
  (if (oddp x)
    (let ((v (* x x)))
      (if v (list v) nil))
    nil))
Nested — unexpanded
(for+ (list x y)
  for x in '(1 2 3)
  for y in '(a b)
  if (oddp x))
Nested — expanded
(loop for x in '(1 2 3)
  append
  (loop for y in '(a b)
    append
    (if (oddp x)
      (let ((v (list x y)))
        (if v (list v) nil))
      nil)))

Source

(defmacro for+ (expr &rest cs)
  (labels ((walk (cs)
             (cond ((null cs)
                    `(let ((v ,expr)) (if v (list v) nil)))
                   ((eq (car cs) 'for)
                    `(loop for ,(cadr cs) in ,(cadddr cs)
                           append ,(walk (cddddr cs))))
                   ((eq (car cs) 'if)
                    `(if ,(cadr cs) ,(walk (cddr cs)) nil)))))
    (walk cs)))

(plus name [doc] slots methods...)

Struct + methods, in one form.

Bundles defstruct with its defmethods. First parameter of every method is i, so $slot just works. Optional doc string. (:opts ((opt ...))) adds defstruct options.

make works like Python's __init__: the body configures the instance; it never returns it. Plus builds i, runs your body, then returns i for you. (make () BODY) forwards keyword args to %make-NAME and binds i automatically. (make (ARGS) BODY) gives custom args; bind i yourself via &aux.

Dispatch: same method name across two plus forms? CL dispatches on the first arg's type. (add s v) hits sym's body or num's depending on s.

Single-impl methods? Don't put them in plus. Make them (defun klass-name (i ...) ...) next to the plus. klass- prefix names ownership; no global generic-function pollution.

Expansion

Unexpanded
(plus sym "Symbolic column."
  ((n 0) has)

  (mid ()
    (car $has)))
Expanded
(progn
  (defstruct (%make-sym)
    "Symbolic column."
    (n 0) has)
  (defun make-sym (&rest args)
    (apply #'%make-sym args))
  (defmethod mid ((i sym))
    (car (slot-value i 'has)))
  'sym)
Post-init make — unexpanded
(plus num "Numeric."
  ((n 0) (mu 0))

  (make ()
    (setf $goal 1)))
Post-init make — expanded
(progn
  (defstruct (%make-num)
    "Numeric."
    (n 0) (mu 0))
  (defun make-num (&rest args)
    (let ((i (apply #'%make-num args)))
      (setf (slot-value i 'goal) 1)
      i))
  'num)

Source

(defmacro plus (name &rest rest)
  (let* ((doc       (when (stringp (car rest)) (pop rest)))
         (slots     (pop rest))
         (methods   rest)
         (extra     (find :opts methods :key #'car))
         (methods   (remove :opts methods :key #'car))
         (make-spec (find 'make methods :key #'car))
         (methods   (remove 'make methods :key #'car))
         (priv      (intern (format nil "%MAKE-~A" name)))
         (pub       (intern (format nil  "MAKE-~A" name)))
         (make-defun
           (cond
             ((null make-spec)
              `(defun ,pub (&rest args)
                 (apply #',priv args)))
             ((null (cadr make-spec))
              `(defun ,pub (&rest args)
                 (let ((i (apply #',priv args)))
                   ,@(cddr make-spec) i)))
             (t
              `(defun ,pub ,(cadr make-spec)
                 ,@(cddr make-spec) i))))
         (opts (cons `(:constructor ,priv)
                     (and extra (cadr extra)))))
    `(progn
       (defstruct (,name ,@opts)
         ,@(when doc (list doc))
         ,@slots)
       ,make-defun
       (plus+ ,name ,@methods)
       ',name)))

(plus+ name methods...)

Add methods to an existing plus later.

Same method shape as plus, no make. Useful when methods need types defined later in the file.

Unexpanded
(plus+ num
  (twice () (* 2 $mu)))
Expanded
(progn
  (defmethod twice ((i num))
    (* 2 (slot-value i 'mu)))
  'num)

(new+ cls k v ...)

Constructor sugar.

Expands to (make-CLS k v ...). Saves the make- prefix at call sites.

Unexpanded
(new+ num :goal 0)
Expanded
(make-num :goal 0)

(defread c (s) body)

Define a reader macro on a single character.

Installs body as the reader macro for character c. The two project reader macros — $ and @ — are built with it.

Unexpanded
(defread @(s)
  `(second
     (assoc ',(read s t nil t)
            *the*)))

@budget
Expanded
(set-macro-character
  #\@
  (lambda (s c)
    (declare (ignore c))
    `(second
       (assoc ',(read s t nil t)
              *the*)))
  t)

(second (assoc 'budget *the*))
