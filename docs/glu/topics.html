Manual

Eleven forms. Each gets a section.

$field

Self slot access.

Inside a method whose first parameter is named i — glu always names it that — $x expands to (slot-value i 'x). Read, setf, incf, push all work.

$mu          ; (slot-value i 'mu)
(incf $n)   ; (incf (slot-value i 'n))
(setf $has nil)

(? x a b c)

Nested slot access.

Chained slot-value. setf-able. Replaces the noisy (slot-value (slot-value ...)) tower.

(? data cols y)
; (slot-value (slot-value data 'cols) 'y)

(setf (? tree left) new-left)

(! f a b)

Funcall shortcut.

Equivalent to (funcall f a b). Three chars shorter, reads cleaner when functions ride in arguments.

(! cmp a b)            ; (funcall cmp a b)
(mapcar (fn (! key _)) lst)

fn / fnn / fnnn

Lambda shortcuts.

One-, two-, three-arg lambdas with implicit parameters _, __, ___. Kills the (lambda (x) ...) overhead in higher-order calls.

(mapcar (fn   (* _ _))     '(1 2 3))     ; (1 4 9)
(mapcar (fnn  (+ _ __))    '(1 2) '(10 20)); (11 22)
(sort lst #'< :key (fn (slot-value _ 'at)))

(aif test then else)

Anaphoric if.

Binds the test result to it in both branches. From Paul Graham's On Lisp.

(aif (find flag opts :key #'third :test #'equalp)
     (setf (second it) (thing arg)))

(aif (gethash key cache)
     it
     (setf (gethash key cache) (compute key)))

(let+ ...)

Sequential bindings: let, dbind, mvb, labels.

One construct, four forms. Entry shape picks: plain (var val) is let; flat-symbol list ((a b c) form) is multiple-value-bind; nested-pattern list is destructuring-bind; 3+ elements (name args body...) is labels.

(let+ ((x        5)                       ; let
       ((a b c)  (values 1 2 3))             ; mvb
       (sq       (n) (* n n))                ; labels
       (((l r) p) '((1 2) 3)))               ; dbind
  (sq (+ x a)))

Replaces nested let* / multiple-value-bind / destructuring-bind / labels stairs with one flat block. let* becomes redundant.

(for/ expr for x in lst if test ...)

List comprehension.

Reads like Python. Walks for and if clauses, collecting expr per surviving iteration. Nil results are skipped — a guard expression doubles as filter.

(for/ (* x x) for x in '(1 2 3 4 5) if (oddp x))
; (1 9 25)

(for/ (list x y)
       for x in '(1 2 3)
       for y in '(a b)
       if (oddp x))
; ((1 A) (1 B) (3 A) (3 B))

(glu name [doc] slots methods...)

Struct + methods, in one form.

Bundles defstruct with its defmethods. First parameter of every method is i, so $slot just works. Optional doc string. The pseudo-method (make ARGS BODY) defines the public factory; (:opts ((opt ...))) adds defstruct options.

(glu sym "Symbolic column."
    ((n 0) has)

  (add (v)
    (incf $n)
    (aif (assoc v $has)
         (incf (cdr it))
         (push (cons v 1) $has)))

  (mid () (car (reduce (fnn (if (> (cdr _) (cdr __)) _ __))
                       $has))))

Dispatch: same method name across two glu forms? CL dispatches on the first arg's type. (add s v) hits sym's body or num's depending on s.

Single-impl methods? Don't put them in glu. Make them (defun klass-name (i ...) ...) next to the glu. klass- prefix names ownership; no global generic-function pollution.

(glu+ name methods...)

Add methods to an existing glu later.

Same method shape as glu, no make. Useful when methods need types defined later in the file.

(glu+ num
  (twice () (* 2 $mu)))

(new cls k v ...)

Constructor sugar.

Expands to (make-CLS k v ...). Saves the make- prefix at call sites.

(new num :goal 0)   ; (make-num :goal 0)

(defread c (s) body)

Define a reader macro on a single character.

Installs body as the reader macro for character c. The two project reader macros — $ and @ — are built with it.

; @key reads as (second (assoc 'key *the*))
(defread @(s)
  `(second (assoc ',(read s t nil t) *the*)))

@budget    ; (second (assoc 'budget *the*))
