## [xy.lisp](xy.lisp)


`add ((c col) x &key (filter #'identity))`

<ul>   Add numbers to column.
</ul>

`add1 ((nu num) x)`

<ul>   New numbers update `min` and `max`.
</ul>

`norm ((n num) x)`

<ul>   Convert x to the range 0..1.
</ul>

`add1 ((s sym) x)`

<ul>   Increment the symbols counts
</ul>

`norm ((s sym) x)`

<ul>   Normalize symbols does nothing.
</ul>
